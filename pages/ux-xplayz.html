01 — Project Goal

목표

유저의 앱 사용 현황과 광고 수익 보상의 수를 한눈에 확인합니다.
유저들간의 랭킹을 보여줌으로써 앱을 활발하게 이용하도록 유도하여, 이용률을 증가 시키고자 합니다.

02 — Problem & Solution

문제 발견 및 솔루션

Problem 01

광고 수익 실시간 확인 불가

유저가 광고를 시청한 후 획득한 수익을 즉시 확인할 수 있는 수단이 없어, 보상 체계에 대한 신뢰도와 참여 동기가 저하됩니다.

Problem 02

모바일 앱 내 유저 간 어뷰징 상시 발생

관리자가 이상 보상 지급 현황을 한눈에 파악하기 어려워, 어뷰징이 탐지되지 않은 채 지속됩니다.

arrow_and_edge
메인 화면

광고 수익 UI

광고 시청으로 얻은 수익 달성률을 IP 별로 메인 화면에 게이지로 표시해, 유저가 보상 현황을 실시간으로 인지할 수 있도록 합니다.

랭킹 화면

보상 이상 탐지 대시보드

랭킹 페이지에 보상 수여 금액을 집계 및 표시하여, 관리자가 비정상적인 지급 패턴을 한눈에 식별할 수 있도록 합니다.

03 — Key Screens

주요 화면

메인 화면

앱에 연결된 실제 사용자의 수와 유저들이 받아간 실제 보상 물량을 실시간으로 데이터를 보여줍니다. 또한, 메인 화면에서 xPlayz 앱의 사용 방법을 확인할 수 있는 상세 페이지도 이어서 확인할 수 있게 웹을 디자인하였습니다.

랭킹 화면

유저들이 xPlayz로 보상 받은 수익을 1일 / 7일 / 30일 간격으로 확인할 수 있는 지표를 작업하였습니다. 관리자가 한 눈에 확인하여 어뷰징 파악을 쉽게 할 수 있도록 디자인 하였습니다.

광고 신청 폼 화면

유저들이 xplayz 앱에 올라갈 광고를 신청할 수 있는 폼 페이지 디자인을 하였습니다. 폼은 자칫 복잡한 화면이 될 수 있기 때문에, 버튼과 텍스트의 배치를 적절히 정리해서 디자인하는 것이 중요합니다. 누구나 쉽게 신청할 수 있도록 디자인하는 것이 목표였습니다.
