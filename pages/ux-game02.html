UX

콘솔 구조 최적화

콘솔(조이콘)의 조작 구조와 가장 자연스러운 배치로 UX를 설계하였습니다.

UI

아트와 조화

힐링 게임에 많은 영향을 주는 아기자기한 그래픽 아트와 조화되는 UI를 제작하였습니다.

02 - DIALOGUE

대화 다이얼로그 - 감성적인 UI

대화 다이얼로그에서 사용하는 조작 프롬프트에 작은 인터랙션 디자인을 적용하고,
전체적인 아트 분위기와 어울리도록 투명 패널과 플랫한 색상을 활용하였습니다.

03 - INVENTORY

인벤토리 - 쉬운 UI

정보가 많고 아이템이 많은 인벤토리의 특성상, UI가 복잡해질 수 있습니다.
그러므로 플레이어가 쉽게 이애하고 조작할 수 있는 화면이 되도록 정보를 최소한으로 보여주었습니다.
